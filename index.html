Atmospheric Electrification Analysis

Integrated Microphysics • Lightning Mapping • Orientation Dynamics

Project Overview

This project presents a data-driven framework for analyzing atmospheric electrification using airborne microphysics, radar observations, and lightning activity.

The system integrates multiple datasets into a unified Electrification Risk Index (ERI) that quantifies storm intensity and charge separation potential.

Project Objectives

Analysis and fusion of synchronized datasets to test the hypothesis that atmospheric electric fields influence the alignment and shape of cloud ice crystals.

Identify microphysical indicators that can be used in early warning systems for lightning hazards.

Data Sources & Instrumentation

This project integrates multiple synchronized datasets collected during airborne observations of convective storms using the NRC Convair-580 research aircraft.

Optical Array Probes (2D-S & HVPS-3)

High-resolution particle imagery is captured using optical array probes, enabling analysis of particle size, shape, and orientation.

Lightning Mapping Array (HLMA)

Lightning activity is obtained from the Houston Lightning Mapping Array, providing spatial and temporal information on flash events.

Radar Reflectivity

Radar data is used to characterize storm intensity and precipitation structure.

Aircraft State Parameters

Aircraft measurements include altitude, temperature, liquid water content, and vertical wind (updraft).
