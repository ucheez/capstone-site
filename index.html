Atmospheric Electrification Detection

HVPS-3 Microphysics • Lightning Mapping • Orientation Analysis

Project Overview

This project investigates atmospheric electrification using airborne microphysics data (HVPS-3) and Lightning Mapping Array (HLMA) observations.

Data Sources

  • HVPS-3 Particle Imaging
  • HLMA Lightning Data
  • F12 Aircraft Flight State Data

NRC Convair-580 Research Aircraft

The NRC Convair-580 is a heavily instrumented atmospheric research aircraft equipped with in-situ and remote sensing systems used for storm electrification and cloud microphysics studies.

Optical Array Probes (2D-S & HVPS-3)

The 2D-S probe captures high-resolution particle imagery for small hydrometeors, while the HVPS-3 extends detection into the millimeter regime for larger aggregates and graupel.

Aircraft Flight Path & Electrification Levels

Flight path colored by flash-rate classification: Low (<5 flashes/min), Elevated (5–10 flashes/min), and High (>10 flashes/min).

2D-S IWC & LWC Time Series

HVPS-3 IWC & LWC Time Series

Concentration & Diameter Time Series

Column Orientation Distribution

Column Orientation Population Over Time

Classification Examples

Strict geometric filtering removes hollow, fragmented, and non-elongated particles while preserving structurally coherent columns.
